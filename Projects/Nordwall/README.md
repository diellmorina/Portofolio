# NordWall Studio Website

A modern single-page business site for facade and exterior services, built with plain HTML, CSS, and JavaScript.

## Features

- Sticky navigation with mobile menu
- Hero background slideshow
- Services and about sections
- Project gallery with show-more toggle
- Lightbox image preview
- Supplier cards
- Contact form with client-side validation feedback
- Responsive layout for desktop and mobile

## Run

Open index.html in a browser.
